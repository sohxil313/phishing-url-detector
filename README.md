# 🔍 Phishing URL Detector

A browser-style URL phishing detector powered by ML.

## 🚀 Features
- Accepts raw URLs
- Feature extraction (HTTPS, IP usage, obfuscation, etc.)
- Random Forest model
- Streamlit frontend
